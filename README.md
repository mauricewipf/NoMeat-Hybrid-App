# NoMeat (Hybrid App)


## Features

- A hybrid, mobile app built on [Ionic Framework](https://ionicframework.com)

## Run locally

```
$ npm install -g cordova ionic	
$ cd NoMeat
$ npm install
$ ionic serve
```

## Deploy to Ionic

```
$ git add .
$ git commit -am "make it better"
$ git push ionic master
```
